# Product Design Direction

## Visual Direction

Theme: fintech bold + data-dense pro

## Principles

1. Mobile-first information hierarchy.
2. High signal, low ornament for dense numeric content.
3. Strong typographic contrast for scanability.
4. Consistent icon language for transaction and category recognition.

## UI Characteristics

1. Tight but readable spacing scale.
2. Compact card system with priority color states.
3. Persistent bottom navigation on small screens.
4. Fast interactions with low-latency transitions.

## Typography and Iconography

1. Use a modern, high-legibility sans family with clear number glyphs.
2. Use tabular numbers in financial tables and KPIs.
3. Use semantic icon sets with color + shape redundancy for accessibility.

## Motion

1. Motion is purposeful: list enter, card reveal, threshold transitions.
2. No decorative heavy animation.
3. Respect reduced motion preferences.

## Accessibility and Responsiveness

1. Minimum AA contrast for all text states.
2. Full keyboard navigation support.
3. Touch targets at least 44x44 px on mobile.
4. Layout tested at 360px, 390px, 768px, and desktop breakpoints.
